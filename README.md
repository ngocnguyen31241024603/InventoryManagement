# Inventory Management

## Mô tả
Dự án quản lý kho được xây dựng bằng ngôn ngữ C#, chạy trên Console Application.

## Chức năng chính
- Thêm, sửa, xoá sản phẩm
- Phân loại sản phẩm theo danh mục
- Quản lý số lượng tồn kho
- Tính giá trị hàng tồn

## Công nghệ sử dụng
- Ngôn ngữ: C#
- Môi trường: .NET Console
